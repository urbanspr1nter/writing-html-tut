Testing HTML for Lesson 22

This page is to test to see if your browser supports the HTML discussed in the lesson for More for Images and Lists. If it does not, then it is probably not worth your time doing this lesson!
No Hypertext Border for Graphic <img... border=0>

If this graphic hyperlink lacks a colored box around it, then your web browser supports supression of the hyperlink indicator for inline images

Bullet Styles for Un-ordered Lists <ul type=xxxx>
  • type=disc is this bullet is a solid round disk?
  • type=square is this bullet marker a square?
  • type=circle is this bullet marker a circle?

Number Styles for Ordered Lists <ol type=xxxx>
type=1 standard "1,2,3..."
  1. item 1
  2. item 2
  3. item 3
type=A "A,B,C..."
  1. item 1
  2. item 2
  3. item 3
type=a "a,b,c..."
  1. item 1
  2. item 2
  3. item 3
type=I "I,II,III..."
  1. item 1
  2. item 2
  3. item 3
type=i "i,ii,iii..."
  1. item 1
  2. item 2
  3. item 3
