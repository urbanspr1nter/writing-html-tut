Lesson 29a Image Download Center

For this lesson, Animate My GIF, you will need to download an animated GIF file and an icon GIF image. To download the images, try one of the methods below.


msh_events.gif (194k)


projector.gif (0.6k)

1. The Secret Menu

Look for the "secret" popup menu in your web browser. (Click and hold the mouse for Macintosh; click and hold the right mouse button for Windows and Unix)

  1. Click and hold the mouse on the image above to bring up the "secret" menu
  2. Select Save this Image as...
  3. When prompted, enter msh_events.gif as the name for the animated GIF file and projector.gif for the icon image. (it may insert it for you)
  4. Save the file in the pictures folder/directory

2. Saving As Source

If you cannot find the secret menu, try this approach. Select the link below ("view the image") that links directly to the image. It will likely show the picture by itself in a separate web page or it may launch an external graphics program to display the image.

  • If the image displays by itself in a web page then:
    1. Select Save As... from the File menu
    2. Select the option to save as SOURCE
    3. Enter msh_events.gif as the name for the animated GIF file and projector.gif for the icon image
    4. Save the file in the pictures folder/directory


  • If the image displays in an external graphics program, then:
    1. Select Save As... from the File menu
    2. Look for an option to save as GIF format
    3. Enter msh_events.gif as the name for the animated GIF file and projector.gif for the icon image
    4. Save the file in the pictures folder/directory
