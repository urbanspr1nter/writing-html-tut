Testing HTML for Lesson 18

This page is to test to see if your browser supports the HTML discussed in the lesson for Spiffing Up text. If it does not, then it is probably not worth your time doing this lesson!
Font Size
If this text is considerably larger than this text

then your web browser supports the <font size=N>...</font> tag

Font Color
If the color of this text is red, then your web browser supports
the <font color=#XXXXXX>...</font> tag

Superscript2
If the "2" appears in a smaller font and elevated position, then your web browser supports the <sup>...</sup> tag.

Subscript2
If the "2" appears in a smaller font and lower position, then your web browser supports the <sub>...</sub> tag.

Font Face
If the text items below appear in different fonts, then your browser supports the <font FACE=XXXX>...</font> tag. If only some of the fonts are different, it may just be that your computer does not have the specified font.
  • Courier
  • Times
  • Arial,Helvetica
  • Monaco
  • Symbol
  • Palatino
  • System,Chicago
