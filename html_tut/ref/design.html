The Seven C's of WebService Design
The University of Georgia provides a concise set of guidelines (with linked examples) that can be used for any organization's web site. The Seven C's include Comprehensiveness, Currentness, Client-Orientation, Clarity over Coolness, Courtesy over Coolness, Compatibility without Compromise, and Cross-Linking and Validation.
http://www.uga.edu/design/
Information Design Tutorial
"is an introduction to the basics of page layout and site design. It assumes that you already have a basic knowledge of HTML...
