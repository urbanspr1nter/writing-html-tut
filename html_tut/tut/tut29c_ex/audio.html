Lesson 29c Sound Studio

For this lesson, Sound of [web] Music, you will need to download the QuickTime movie of a person reading a letter.


363k QuickTime sound

If you do not see the movie or you get an error message about a missing plug-in, you can go to Apple's web site to download the latest version of QuickTime. If you do see the movie, you can play it here to verify that it works (the play button is the small triangle button.

To save this movie so you can use it in your lesson:

  1. Access the "secret" popup menu (click and hold the mouse for Macintosh; click and hold the right mouse button for Windows and Unix) for this link to the movie. Do not click and follow the link!
    link to movie

  2. Select Save this Link as... or Save Target as... from the "secret" popup menu.
  3. If there is a menu labeled Format, select Source.
  4. Save the movie file named pliny.mov inside your movies folder/directory.

Once you have downloaded and saved this file, return to lesson 29c
