Downloading Image Files

Using the "secret" pop-up menu
  1. For Macintosh, click the mouse down on the desired image and hold it until a pop-up menu appears. For Windows and Unix, click the mouse down with the right mouse button to see the menu.
  2. Select Save this Image As... from the pop-up menu.
  3. When prompted, save the file with an appropriate name to the desired location on your computer's hard drive.
With your web browser, you can save a file to the local hard drive. The exact steps or menu names will depend on which application you are using.
