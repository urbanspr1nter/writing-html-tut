Testing HTML for Lesson 16

This page is to test to see if your browser supports the HTML discussed in the lesson for Colorful and Textured Backgrounds. If it does not, then it is probably not worth your time doing this lesson!

Background Colors
If the color of the previous page is a light BLUE then your browser supports solid color backgrounds.
Text Colors
If the color of the text of this page is Red-Brown then your browser supports colored text from the <body> tag.
HyperText Colors
If the color of the text of this is GREEN then your browser supports colored hypertext from the <body> tag
Visited HyperText Colors
If the color of the text of this link is PINK then your browser supports colored visited hypertext from the <body> tag.
Textured Backgrounds
If the background of this page has a light brown "paper bag" texture, then your browser supports background files from the <body> tag.
